Movie Quiz

This is a movie quiz static web application.
For each movie display you will have to guess the director or one of the actors of the movie.
Each answer must be a full name !
If you find an actor or a director --> you will have informations about him and will have to find a movie where he was
If you don't find --> you will have to retry with another movie
A movie can't be used twice !

That was all the instructions, I hope you will have some fun using this app !
Good Luck !
